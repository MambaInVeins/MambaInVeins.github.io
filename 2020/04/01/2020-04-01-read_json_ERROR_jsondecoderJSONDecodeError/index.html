读取json文件报错:json.decoder.JSONDecodeError

针对安全客API使用过程中，对https://api.anquanke.com/data/v1/posts?page=2&size=10&category=news&category=news提供的json文件进行json.load()出错。

Extra data: line 1 column 4 (char 3)

出错原因：

读取json文件数据时，由于指定的json文件内包含多个json格式的数据，所以报错。

Invalid control character at: line 2 column 18 (char 19)

出错原因：

json默认使用的是严谨格式，当跨语言传递数据时，就容易报出这个错误。

解决方法：

加上参数 strict
