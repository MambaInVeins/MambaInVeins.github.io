ubuntu git安装及使用

一、git安装

首先需要注册一个全球最大同性交友网站github账号

打开终端，安装git

配置git账号

根据用户名及邮箱生成密钥(该密钥会用在该账号中)

查看刚刚目录下的密钥

复制密钥,注意别复制邮箱

打开https://github.com/网站,登陆你的账号，进入settings-SSH and GPG keys

输入个标题,然后复制粘贴你的密钥

检测是否可用ssh -T git@github.com

这就表示成功了，就可以在本机上传及更新github项目。

二、git上传

在github中新建项目

在本地创建一个项目目录并进入

当我们需要给新的项目respository上传内容时，需要使用init命令将我们需要上传的目录初始化

链接远程仓库: git remote add origin git@github.com:用户名/新建github项目名.git

在本地项目目录中执行命令，将需要上传到远程仓库的文件写在add后面

将add后的文件提交到本地仓库 git commit -m “提交的信息”

将提交的内容同步至github上

这里可能会遇到问题：

问题1：git上传位置出错

解决:修改配置文件

1.进入当前共享文件的 .git 目录    cd .git

2.编辑 config      vim config

3.将url地址改成github中项目的 ssh地址

4.保存并退出

5.再次提交, 不需要输入账号密码)

问题2：新建项目时选择了新建README.md文件，而github中的README.md文件不在本地代码目录中

解决:先pull README.md文件

解决问题后，再执行

测试文件成功上传至github。

三、总结上传步骤

在当前项目的目录中生成本地的git管理库，我们可以发现在当前目录下多了一个.git的隐藏文件夹。

查看能被上传的文件

将项目中所有的文件添加到仓库中，如果有的文件之前已经添加了，它会自动省略。如果想添加特定文件，把.换成对应的文件名就行了。

本次提交的注释，双引号中的字符自定义修改。git commit 后面加 -m 表示可以直接输入提交说明，如果不加 -m ，直接输入git commit，就会弹出一个类似于 vim 的界面，让你输入提交说明。

新建一个github的仓库，在网页新建，记下地址，也就是仓库的地址，输入命令链接远程仓库
