ubuntu server 18.04 运行 flask

1.安装ubuntu server 18.04

(1)换源:我自己换成了阿里源，换完之后记得更新

(2)安装python3 pip3 以及nginx,mysql

python3一般都自带了

其他的sudo apt-get install python3-pip nginx mysql-server

PS：如果flask需要数据库则安装，不需要跳过

mysql需要配置，可自行百度或者查看之前的文章

《Ubuntu18.04 安装MySQL》

《Mysql 只能sudo连接的问题》

2.利用xshell的xftp传输文件

主要是flask web程序/数据库sql

3.导入数据库

4.配置flaskweb所需包，uwsgi，nginx

将nginx配置写入/etc/nginx/nginx.conf

将uwsgi启动命令写入shell脚本，chmod赋予权限

放置在 /etc/profile.d/ 开机自启动
