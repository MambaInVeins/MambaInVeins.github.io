C语言题目-古风排版

题目描述：

小甲正在研究一种古风排版。

众所周知，我国的古人写字使用的是从右向左的竖向排版，例如wo ai zhong guo按照每列5个字符排版就应该写成如下格式：

 zw
gho
uo 
ona
 gi

注意从右向左的最后一列如果不足字符，应该补齐空格。

现在有人告诉你每列的字符数，小甲希望你把所有字符串按照古风格式转换。

输入格式

第一行一个整数k，表示古风格式每列k（1<k<100）个字符。

第二行一个字符，仅包含大小写字符，空格，和英文感叹号！。为了方便输入，第二行的字符串结尾没有换行符（即没有’\r’或’\n’）。

保证字符串长度不超过1000。

输出格式

输出若干行，每列k个字符，表示转换为古风格式的结果。

输入样例

5

wo aizhong guo

输出样例

 zw
gho
uo 
ona
 gi

程序代码

// 注意事项：1. 读入的时候如果用gets最好判断一下'\r'(本题数据比较友好，实测没有\r卡大家)
// 2. 最左边一列没摆满务必补齐空格，实现方式有很多，可以在数组里直接补上空格，或者是二重循环一个一个字符输出，然后遇到\0空字符就输出空格
#include <stdio.h>
#include <string.h>
char gf[1002][1002];
char txt[1005];
int l, k;
void display()
{
    int i;
    for (i = 0; i < k; i++)
        printf("%s\n", gf[i]);
}
int main()
{
    int i, j, p = 0;
    char c;
    scanf("%d", &k);
    while (getchar() != '\n');
    gets(txt);
    l = strlen(txt);
    if (txt[l - 1] == '\r') txt[--l] = 0;
    for (j = l / k + (l % k != 0) - 1, i = 0; p < l; p++)
    {
        gf[i][j] = txt[p];
        i++;
        if (i == k)i = 0, j--;
    }
    for (i = 0; i < k; i++)
        if (gf[i][j] == 0)gf[i][j] = ' ';
    for (i = 0; i < k; i++)
        printf("%s\n", gf[i]);
    return 0;
}
