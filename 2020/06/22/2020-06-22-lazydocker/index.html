Docker 管理工具 lazydocker

发表于2020-06-22|更新于2020-12-31|docker
|字数总计:127|阅读时长:1分钟|阅读量:

传送门

lazydocker，一个简单的 docker 和 docker-compose 终端用户界面，目前有15.3k star。

支持的系统：Linux, OSX, Windows

功能特性：

鼠标、键盘都可操作
快速查看 docker/docker-compose 容器环境，一目了然
查看 container/service 的日志
容器指标可视化
自己配置你关注的各项指标图形
containers/services 重启、删除、重新构建
清理无用镜像、容器、数据卷

ubuntu安装

curl https://raw.githubusercontent.com/jesseduffield/lazydocker/master/scripts/install_update_linux.sh | bash

使用

lazydocker
