目录爆破工具——dirsearch

发表于2020-08-05|更新于2021-09-15|安全基础知识web
|字数总计:231|阅读时长:1分钟|阅读量:

dirsearch是一个基于python的命令行工具，旨在暴力扫描页面结构，包括网页中的目录和文件。

特点

  • 可保持连接
  • 支持多种后缀（-e|–extensions asp,php）
  • 生成报告（纯文本，JSON）
  • 启发式检测无效的网页
  • 递归的暴力扫描
  • 支持HTTP代理
  • 用户代理随机化
  • 批量处理
  • 请求延迟
  • 扫描器与字典

用法示例

例如

python dirsearch.py -u 10.0.3.45 -e php
python dirsearch.py -u http://220.249.52.133:41417 -e *

常用参数

-u 指定url
-e 指定网站语言
-w 可以加上自己的字典（带上路径）
-r 递归跑（查到一个目录后，在目录后在重复跑，很慢，不建议用）
--random-agents 使用代理（使用的代理目录在uesr-agents.txt中，可以自己添加）

db目录是此款工具自带的字典，reports目录是你所有爬过的网站
