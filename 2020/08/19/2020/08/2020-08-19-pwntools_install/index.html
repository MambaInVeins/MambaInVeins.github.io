安装pwntools

发表于2020-08-19|更新于2021-05-25|ctfpwn
|字数总计:173|阅读时长:1分钟|阅读量:

python2安装pwntools

更新pip

python -m pip install --upgrade pip -i https://pypi.douban.com/simple/

安装pwntools

pip install --upgrade pwntools -i https://pypi.douban.com/simple/

遇到问题：

Installing collected packages: capstone, ropgadget, psutil, intervaltree, pyelftools, pwntools
Attempting uninstall: capstone
    Found existing installation: capstone 3.0.4
ERROR: Cannot uninstall 'capstone'. It is a distutils installed project and thus we cannot accurately determine which files belong to it which would lead to only a partial uninstall.

跳过capstone包安装

pip install --upgrade pwntools --ignore-installed capstone -i https://pypi.douban.com/simple/

测试

python
from pwn import *
asm("xor eax,eax")

python3安装pwntools

cd ~
git clone https://github.com/Gallopsled/pwntools
cd pwntools

apt-get install python3 python3-pip python3-dev git libssl-dev libffi-dev build-essential
pip3 install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
python3 setup.py install
#验证方法
python3
import pwn
pwn.asm("xor eax,eax")
