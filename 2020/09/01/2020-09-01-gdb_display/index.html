GDB自动显示display

发表于2020-09-01|更新于2020-12-31|debug
|字数总计:681|阅读时长:2分钟|阅读量:

使用 GDB 调试程序，可能会遇到某一个表达式需要频繁打印，例如程序在运行时，需要观察某一个表达式值的变化，每次运行程序都需要自己打印，会增加调试程序的负担，所以 GDB 中提供了自动显示表达式的命令。GDB 自动显示使用到 display 命令，命令的详细介绍如下。

设置表达式自动显示

与打印表达式的值 print 命令的使用方式相同，相关的命令格式如下：

display [format] [experssion]

format 表示命令自动显示的格式，experssion 表示程序中的表达式。该命令表示将表达式放入自动显示列表中，当程序运行到断点处时，表达式就会以 format 的格式输出。实例：

(gdb) display i             //自动显示变量 i 的值。
(gdb) display /t i         //以 2 进制的格式自动显示变量的值。

设置内存数据自动显示

与查看内存 x 命令的使用方式相同，相关的命令格式如下：

display [format] [addr]

format 为显示内存的格式，addr 表示内存中的一个地址。该命令表示将 addr 放入自动显示的列表中，程序中断时查看内存中的数据。实例：

(gdb) display /x main             //以 16 进制的格式打印输出 main 函数所在内存地址中的数据。
(gdb) display /x 0x1234         //以 16 进制的格式打印输出 0x1234 所在内存地址中的数据。

查看自动显示列表

查看 GDB 中的自动显示列表使用命令：

info display

该命令会将加入自动显示列表的表达式或者是地址显示出来。

设置自动显示选项

加入自动显示列表中的表达式或者是地址，GDB 会给它分配一个编号，编号一般从 1 开始。我们可以根据这个编号删除或禁用这个选项。

  1. 删除自动显示中的选项，命令格式如下：

    undisplay num
    delete display num

num 表示自动显示列表的标号。

  1. 禁用或激活自动显示列表中的选项，命令格式如下：

    disable display [num] //禁用自动显示选项
    enable display [num] //激活自动显示选项

num 表示自动显示列表中的标号。禁用后的选项在程序中断后不会自动输出，但是系统会记录这个选项，在后期可以激活。

常用

每次程序中断后可以看到即将被执行的下一条汇编指令，其中 $pc 代表当前汇编指令，/i 表示以十六进行显示。

display /i $pc

查看ebx寄存器

display /x $ebx
